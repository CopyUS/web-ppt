# Web.PPT — Pages web en direct dans PowerPoint

> Affichez n'importe quel site web pendant votre presentation PowerPoint — entierement controlable avec une telecommande, sans jamais quitter le diaporama.

---

## Pourquoi Web.PPT ?

| Outil | Probleme |
|-------|----------|
| **LiveWeb VBA** | Utilisait le controle ActiveX d'Internet Explorer — abandonne par Microsoft. Capturait egalement le focus clavier/souris, rendant impossible le controle de la presentation avec une telecommande. |
| **Microsoft Web Viewer** | Ne fonctionne pas en mode diaporama ([bugs connus](https://github.com/OfficeDev/office-js/issues/4410), non corriges depuis des annees) |
| **ClassPoint embedded browser** | Supprime en octobre 2025 |

**Web.PPT** est un Office Add-in gratuit et open-source qui comble cette lacune. Il fonctionne partout ou PowerPoint est disponible — Windows, Mac, iPad et Web.

### Comment ca fonctionne

1. Vous saisissez une URL dans le panneau de configuration (Task Pane)
2. Pendant le diaporama, vous cliquez sur **Show WebPage.PPT** dans le ruban
3. Une fenetre de navigateur s'ouvre par-dessus votre presentation avec le site web en direct
4. Le minuteur de fermeture automatique ferme la fenetre du viewer apres le delai defini, rendant le focus a PowerPoint — vous pouvez ainsi continuer a naviguer entre les diapositives avec une telecommande

Pas de macros VBA. Pas d'ActiveX. Aucune difficulte d'installation.

---

## Configuration requise

| Exigence | Details |
|----------|---------|
| **Version de PowerPoint** | **Microsoft 365** (abonnement) ou **Office 2021** (achat unique) |
| **Systeme d'exploitation** | Windows 10/11, macOS, iPadOS, ou tout OS disposant d'un navigateur (PowerPoint Online) |
| **Connexion Internet** | Requise pour charger les pages web |

### Pourquoi Office 2019 et les versions anterieures ne sont pas prises en charge

Web.PPT utilise l'API JavaScript de PowerPoint (`PowerPoint.run`, `getSelectedSlides`) pour detecter la diapositive active et enregistrer les parametres par diapositive. Cette API (requirement set **PowerPointApi 1.1+**) n'est disponible que dans :

- Microsoft 365 (tous les canaux de mise a jour)
- Office 2021 (licence perpetuelle)

Office 2019, 2016 et les versions anterieures n'incluent pas ces API, de sorte que l'add-in **ne peut pas determiner quelle diapositive est active** et ne fonctionnera pas correctement.

## Installation

### Option 1 : AppSource (Microsoft 365)

> Cette option sera eventuellement disponible apres la publication de l'add-in sur AppSource.

Fonctionne sur toutes les plateformes (Windows, Mac, iPad, Web) :

1. Ouvrez PowerPoint → onglet **Insertion** → **Obtenir des complements**
2. Recherchez **"Web.PPT"**
3. Cliquez sur **Ajouter**

---

### Option 2 : Sideloading — PowerPoint 2021 / 365 Desktop (Windows)

#### Etape 1 : Activer l'onglet Developpeur dans le ruban

1. Ouvrez PowerPoint → **Fichier** → **Options** → **Personnaliser le ruban**
2. Dans la colonne de droite, cochez la case **Developpeur**
3. Cliquez sur **OK**

#### Etape 2 : Telecharger le manifest

Telechargez `manifest-gh.xml` depuis la [page des versions](https://github.com/CopyUS) ou depuis le dossier `dist/`.

#### Etape 3 : Creer un dossier partage

1. Creez un dossier : `C:\OfficeAddins\`
2. Placez `manifest-gh.xml` dans ce dossier
3. Clic droit → **Proprietes** → **Partage** → **Partager...**
4. Partagez avec vous-meme, copiez le **chemin reseau** (par ex., `\\VOTREPC\OfficeAddins`)

#### Etape 4 : Ajouter le dossier comme catalogue approuve

1. PowerPoint → **Fichier** → **Options** → **Centre de gestion de la confidentialite** → **Parametres du Centre de gestion de la confidentialite...**
2. **Catalogues de complements approuves** → collez le chemin reseau → **Ajouter un catalogue** → cochez **Afficher dans le menu**
3. **OK** → **OK** → **Redemarrez PowerPoint**

#### Etape 5 : Charger l'add-in

1. PowerPoint → **Insertion** → **Obtenir des complements** (ou **Mes complements**) → onglet **Dossier partage**
2. Trouvez **Web.PPT** → **Ajouter**

---

### Option 3 : Sideloading — PowerPoint Online

1. [office.com](https://www.office.com/) → PowerPoint → ouvrez une presentation
2. **Insertion** → **Complements** → **Charger mon complement** → selectionnez `manifest.json` → **Charger**

> Persiste uniquement pour la session en cours.

---

### Option 4 : Sideloading — Mac (Desktop)

1. Telechargez `manifest-gh.xml`
2. Copiez dans `~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef/`
3. Redemarrez PowerPoint → **Insertion** → **Mes complements** → **Mon organisation**

---

## Vider le cache Office (apres les mises a jour)

### Windows
1. Fermez toutes les applications Office
2. Naviguez vers `%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\`
3. Supprimez tout le contenu, redemarrez PowerPoint

### Mac
1. Fermez toutes les applications Office
2. Supprimez le contenu de `~/Library/Containers/com.microsoft.Powerpoint/Data/Library/Caches/`
3. Recopiez le manifest dans le dossier wef, redemarrez PowerPoint

### PowerPoint Online
1. F12 → Application → Storage → Clear site data → rafraichissez la page

## Utilisation

Chaque diapositive peut avoir sa propre URL. Selectionnez une diapositive, ouvrez le Task Pane (**Insertion** → **Add WebPage.PPT**).

| Parametre | Description | Par defaut |
|-----------|-------------|------------|
| **URL** | Adresse web | *(aucune)* |
| **Zoom** | 50%–300% | 100% |
| **Largeur de fenetre** | 30%–100% | 80% |
| **Hauteur de fenetre** | 30%–100% | 80% |
| **Ouverture automatique** | Ouvrir au changement de diapositive | Desactive |
| **Fermeture automatique** | Fermer apres un delai (0–3600s) | Desactive |

10 langues prises en charge. Menu deroulant de langue dans le Task Pane.

## Pour vos propres sites

Ajoutez une seule ligne a la configuration de votre serveur :

**Nginx**
```nginx
add_header Content-Security-Policy "frame-ancestors *";
```

**Apache**
```apache
Header set Content-Security-Policy "frame-ancestors *"
Header unset X-Frame-Options
```

**Node.js / Express**
```javascript
app.use((req, res, next) => {
  res.setHeader('Content-Security-Policy', 'frame-ancestors *');
  res.removeHeader('X-Frame-Options');
  next();
});
```

## Limitations connues

- Les sites bloquant les iframes → "Ouvrir directement"
- PowerPoint Online : pas d'ouverture automatique au changement de diapositive
- Office 2019 et versions anterieures : non pris en charge
- Le viewer s'ouvre dans une fenetre de navigateur separee
