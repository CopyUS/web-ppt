# Web.PPT — Páginas web ao vivo no PowerPoint

> Mostre qualquer site durante sua apresentação no PowerPoint — totalmente controlável com um apresentador, sem jamais sair do modo de apresentação.

---

## Por que Web.PPT?

| Ferramenta | Problema |
|------------|----------|
| **LiveWeb VBA** | Usava o controle ActiveX do Internet Explorer — descontinuado pela Microsoft. Além disso, capturava o foco do teclado/mouse, tornando impossível controlar a apresentação com um clicker ou controle remoto. |
| **Microsoft Web Viewer** | Não funciona no modo de apresentação ([bugs conhecidos](https://github.com/OfficeDev/office-js/issues/4410), sem correção há anos) |
| **Navegador integrado ClassPoint** | Removido em outubro de 2025 |

**Web.PPT** é um Suplemento do Office gratuito e de código aberto que preenche essa lacuna. Funciona em todos os lugares onde o PowerPoint roda — Windows, Mac, iPad e Web.

### Como funciona

1. Você insere uma URL no painel de configurações (Task Pane)
2. Durante a apresentação, você clica em **Show WebPage.PPT** na faixa de opções
3. Uma janela do navegador abre sobre sua apresentação com o site ao vivo
4. O temporizador de fechamento automático fecha a janela do visualizador automaticamente após o tempo definido, devolvendo o foco ao PowerPoint — para que você possa continuar navegando pelos slides com um clicker ou controle remoto

Sem macros VBA. Sem ActiveX. Sem dores de cabeça com instalação.

## Requisitos do sistema

| Requisito | Detalhes |
|-----------|----------|
| **Versão do PowerPoint** | **Microsoft 365** (assinatura) ou **Office 2021** (compra única) |
| **Sistema operacional** | Windows 10/11, macOS, iPadOS ou qualquer SO com navegador (PowerPoint Online) |
| **Conexão com a internet** | Necessária para carregar páginas web |

### Por que o Office 2019 e versões anteriores não são suportados

O Web.PPT utiliza a API JavaScript do PowerPoint (`PowerPoint.run`, `getSelectedSlides`) para detectar o slide atual e salvar configurações por slide. Essa API (conjunto de requisitos **PowerPointApi 1.1+**) está disponível apenas no Microsoft 365 e no Office 2021. O Office 2019/2016 e versões anteriores não incluem essas APIs.

## Instalação

### Opção 1: AppSource (Microsoft 365)
> Esta opção possivelmente estará disponível após a publicação do suplemento no AppSource.

1. Abra o PowerPoint → aba **Inserir** → **Obter Suplementos**
2. Pesquise por **"Web.PPT"**
3. Clique em **Adicionar**

### Opção 2: Sideloading — PowerPoint 2021 / 365 Desktop (Windows)

#### Etapa 1: Habilitar a guia Desenvolvedor
1. PowerPoint → **Arquivo** → **Opções** → **Personalizar Faixa de Opções**
2. Marque **Desenvolvedor** → **OK**

#### Etapa 2: Baixar o manifesto
Baixe o `manifest-gh.xml` em [releases](https://github.com/CopyUS) ou na pasta `dist/`.

#### Etapa 3: Criar uma pasta compartilhada
1. Crie `C:\OfficeAddins\`, coloque o `manifest-gh.xml` lá
2. Clique com botão direito → Propriedades → Compartilhamento → Compartilhar
3. Copie o caminho de rede (ex.: `\\SEUPC\OfficeAddins`)

#### Etapa 4: Adicionar como Catálogo Confiável
1. PowerPoint → **Arquivo** → **Opções** → **Central de Confiabilidade** → **Configurações da Central de Confiabilidade...**
2. **Catálogos de Suplementos Confiáveis** → cole o caminho → **Adicionar catálogo** → marque **Mostrar no Menu**
3. OK → OK → **Reinicie o PowerPoint**

#### Etapa 5: Carregar o suplemento
1. **Inserir** → **Obter Suplementos** → aba **Pasta Compartilhada** → **Web.PPT** → **Adicionar**

### Opção 3: Sideloading — PowerPoint Online
1. [office.com](https://www.office.com/) → PowerPoint → apresentação
2. **Inserir** → **Suplementos** → **Carregar Meu Suplemento** → selecione `manifest.json` → **Carregar**
> Persiste apenas para a sessão atual.

### Opção 4: Sideloading — Mac (Desktop)
1. Copie `manifest-gh.xml` para `~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef/`
2. Reinicie o PowerPoint → **Inserir** → **Meus Suplementos** → **Minha Organização**

## Limpando o cache do Office (após atualizações)

### Windows
1. Feche todos os aplicativos do Office
2. Navegue até `%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\`
3. Exclua todo o conteúdo, reinicie o PowerPoint

### Mac
1. Feche todos os aplicativos do Office
2. Exclua o conteúdo de `~/Library/Containers/com.microsoft.Powerpoint/Data/Library/Caches/`
3. Copie novamente o manifesto para a pasta wef, reinicie

### PowerPoint Online
F12 → Application → Storage → Clear site data → atualize a página

## Uso

Cada slide pode ter sua própria URL. Abra o Task Pane (**Inserir** → **Add WebPage.PPT**).

| Configuração | Descrição | Padrão |
|--------------|-----------|--------|
| **URL** | Endereço web | *(nenhum)* |
| **Zoom** | 50%–300% | 100% |
| **Largura da janela** | 30%–100% | 80% |
| **Altura da janela** | 30%–100% | 80% |
| **Abertura automática** | Abrir ao mudar de slide | Desligado |
| **Fechamento automático** | Fechar após tempo (0–3600s) | Desligado |

10 idiomas suportados. Menu suspenso de idioma no Task Pane.

## Para seus próprios sites

**Nginx**
```nginx
add_header Content-Security-Policy "frame-ancestors *";
```

**Apache**
```apache
Header set Content-Security-Policy "frame-ancestors *"
Header unset X-Frame-Options
```

**Node.js / Express**
```javascript
app.use((req, res, next) => {
  res.setHeader('Content-Security-Policy', 'frame-ancestors *');
  res.removeHeader('X-Frame-Options');
  next();
});
```

## Limitações conhecidas
- Sites que bloqueiam iframes → "Abrir diretamente"
- PowerPoint Online: sem abertura automática
- Office 2019 e versões anteriores: não suportados
- Visualizador em janela separada
