# Web.PPT — Paginas web en vivo en PowerPoint

> Muestra cualquier sitio web durante tu presentacion de PowerPoint — totalmente controlable con un clicker, sin salir nunca de la presentacion.

---

## Por que Web.PPT?

Mostrar una pagina web en vivo durante una presentacion deberia ser sencillo. Desafortunadamente, no lo es:

| Herramienta | Problema |
|-------------|----------|
| **LiveWeb VBA** | Usaba el control ActiveX de Internet Explorer — descontinuado por Microsoft. Ademas captura el foco del teclado/raton, lo que hace imposible controlar la presentacion con un clicker o control remoto. |
| **Microsoft Web Viewer** | No funciona en modo de presentacion ([errores conocidos](https://github.com/OfficeDev/office-js/issues/4410), sin corregir durante anos) |
| **ClassPoint embedded browser** | Eliminado en octubre de 2025 |

**Web.PPT** es un complemento de Office gratuito y de codigo abierto que llena este vacio. Funciona en todas las plataformas donde se ejecuta PowerPoint — Windows, Mac, iPad y Web.

### Como funciona

1. Introduces una URL en el panel de configuracion (Task Pane)
2. Durante la presentacion, haces clic en **Show WebPage.PPT** en la cinta de opciones
3. Se abre una ventana del navegador sobre tu presentacion con el sitio web en vivo
4. El temporizador de cierre automatico cierra la ventana del visor automaticamente despues del tiempo establecido, devolviendo el foco a PowerPoint — para que puedas continuar navegando por las diapositivas con un clicker o control remoto

Sin macros VBA. Sin ActiveX. Sin problemas de instalacion.

## Requisitos del sistema

| Requisito | Detalles |
|-----------|----------|
| **Version de PowerPoint** | **Microsoft 365** (suscripcion) o **Office 2021** (compra unica) |
| **Sistema operativo** | Windows 10/11, macOS, iPadOS, o cualquier SO con navegador (PowerPoint Online) |
| **Conexion a Internet** | Necesaria para cargar paginas web |

### Por que Office 2019 y versiones anteriores no son compatibles

Web.PPT utiliza la API JavaScript de PowerPoint (`PowerPoint.run`, `getSelectedSlides`) para detectar la diapositiva actual y guardar la configuracion por diapositiva. Esta API (conjunto de requisitos **PowerPointApi 1.1+**) solo esta disponible en:

- Microsoft 365 (todos los canales de actualizacion)
- Office 2021 (licencia perpetua)

Office 2019, 2016 y versiones anteriores no incluyen estas API, por lo que el complemento **no puede determinar cual diapositiva esta activa** y no funcionara correctamente.

## Instalacion

### Opcion 1: AppSource (Microsoft 365)

> Esta opcion estara posiblemente disponible despues de que el complemento se publique en AppSource.

Funciona en todas las plataformas (Windows, Mac, iPad, Web):

1. Abre PowerPoint → pestana **Insertar** → **Obtener complementos**
2. Busca **"Web.PPT"**
3. Haz clic en **Agregar**

---

### Opcion 2: Sideloading — PowerPoint 2021 / 365 Desktop (Windows)

Usa este metodo para instalar desde un archivo local o carpeta compartida. Es el escenario basico cuando AppSource no esta disponible.

#### Paso 1: Habilitar la pestana Desarrollador en la cinta de opciones

La pestana Desarrollador es necesaria para Sideloading de complementos. Si no la ves:

1. Abre PowerPoint → **Archivo** → **Opciones** → **Personalizar cinta de opciones**
2. En la columna derecha ("Pestanas principales"), marca la casilla **Desarrollador**
3. Haz clic en **Aceptar**

#### Paso 2: Descargar el manifest

Descarga `manifest-gh.xml` desde la [pagina de lanzamientos](https://github.com/CopyUS) o desde la carpeta `dist/`.

#### Paso 3: Crear una carpeta compartida

1. Crea una carpeta, por ejemplo: `C:\OfficeAddins\`
2. Coloca `manifest-gh.xml` en esta carpeta
3. Clic derecho → **Propiedades** → pestana **Compartir** → **Compartir...**
4. Comparte contigo mismo
5. Copia la **ruta de red** (por ejemplo, `\\TUPC\OfficeAddins`)

#### Paso 4: Agregar la carpeta como Catalogo de confianza

1. PowerPoint → **Archivo** → **Opciones** → **Centro de confianza** → **Configuracion del Centro de confianza...**
2. **Catalogos de complementos de confianza** en el menu izquierdo
3. Pega la ruta de red → **Agregar catalogo**
4. Marca **Mostrar en el menu**
5. **Aceptar** → **Aceptar**
7. **Reinicia PowerPoint**

#### Paso 5: Cargar el complemento

1. Reinicie PowerPoint (ciérrelo completamente y vuelva a abrirlo)
2. Vaya a la pestaña **Desarrollador** → haga clic en **Complementos** (Add-ins)
3. Seleccione la pestaña **Carpeta compartida** (Shared Folder) en la parte superior
4. Verá **Web.PPT** — haga clic en **Agregar** (Add)
5. Espere unos segundos — los botones **Add WebPage.PPT** y **Show WebPage.PPT** aparecerán en la pestaña **Insertar**

> **Importante:** Los botones **no aparecen automáticamente** en la cinta después de agregar la carpeta de confianza. Debe cargar manualmente el complemento a través de **Desarrollador → Complementos → Carpeta compartida**. Esta es una característica de la instalación por sideloading — necesita agregar el complemento nuevamente cada vez que reinicie PowerPoint. Después de la publicación en AppSource, el complemento se cargará automáticamente.

#### Si algo salió mal

| Problema | Solución |
|----------|----------|
| El complemento no aparece en "Carpeta compartida" | Verifique la ruta de red en la configuración de catálogos de confianza. Asegúrese de que **"Mostrar en el menú"** esté marcado. Cierre y vuelva a abrir PowerPoint completamente. |
| La pestaña "Carpeta compartida" está vacía | Cierre **todas** las ventanas de PowerPoint, espere 10 segundos y vuelva a abrir. A veces se necesitan 2-3 reinicios. |
| "Add-in Error" al hacer clic en un botón | Limpie la caché de Office (consulte la sección de limpieza de caché a continuación), reinicie PowerPoint y agregue el complemento nuevamente. |
| Los botones aparecieron pero no responden | Espere 5-10 segundos después de cargar — Office descarga archivos del servidor. Verifique su conexión a internet. |

## Limpiar la cache de Office (despues de actualizaciones)

### Windows
1. Cierra todas las aplicaciones de Office
2. Navega a `%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\`
3. Elimina todo el contenido
4. Reinicia PowerPoint

### Mac
1. Cierra todas las aplicaciones de Office
2. Elimina el contenido de `~/Library/Containers/com.microsoft.Powerpoint/Data/Library/Caches/`
3. Vuelve a copiar el manifest a `~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef/`
4. Reinicia PowerPoint

### PowerPoint Online
1. F12 → Application → Storage → Clear site data
2. Actualiza la pagina y vuelve a cargar el manifest

## Uso

### Configurar una URL para una diapositiva

Cada diapositiva puede tener su propia URL. Selecciona la diapositiva, abre el Task Pane (**Insertar** → **Add WebPage.PPT**), introduce la URL.

| Configuracion | Descripcion | Valor predeterminado |
|---------------|-------------|----------------------|
| **URL** | Direccion web | *(ninguna)* |
| **Zoom** | Escala (50%–300%) | 100% |
| **Ancho de ventana** | 30%–100% de la pantalla | 80% |
| **Alto de ventana** | 30%–100% de la pantalla | 80% |
| **Apertura automatica** | Abrir al cambiar de diapositiva | Desactivado |
| **Cierre automatico** | Cerrar despues de un tiempo (0–3600s) | Desactivado |

### Mostrar una pagina web

1. Inicia la presentacion (**F5**)
2. Haz clic en **Show WebPage.PPT**
3. Cierra la ventana o espera al cierre automatico

### Cambiar idioma

10 idiomas disponibles. Menu desplegable de idioma en el encabezado del Task Pane — actualizacion instantanea.

## Para tus propios sitios

Agrega una linea a la configuracion del servidor:

**Nginx**
```nginx
add_header Content-Security-Policy "frame-ancestors *";
```

**Apache**
```apache
Header set Content-Security-Policy "frame-ancestors *"
Header unset X-Frame-Options
```

**Node.js / Express**
```javascript
app.use((req, res, next) => {
  res.setHeader('Content-Security-Policy', 'frame-ancestors *');
  res.removeHeader('X-Frame-Options');
  next();
});
```

No tienes acceso al servidor? Haz clic en **"Open directly"** en el visor.

## Limitaciones conocidas

- Sitios que bloquean iframes → usa "Open directly"
- PowerPoint Online: sin apertura automatica al cambiar de diapositiva
- Office 2019 y versiones anteriores: no compatibles
- El visor se abre en una ventana separada (limitacion de la plataforma)
