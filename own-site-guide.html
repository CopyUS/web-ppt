How to allow embedding

Add one of these snippets to your server config: 

add_header Content-Security-Policy "frame-ancestors *";
Header set Content-Security-Policy "frame-ancestors *"
Header unset X-Frame-Options
app.use((req, res, next) => {
  res.setHeader('Content-Security-Policy', 'frame-ancestors *');
  res.removeHeader('X-Frame-Options');
  next();
});
<meta http-equiv="Content-Security-Policy"
      content="frame-ancestors *">

Note: frame-ancestors in a meta tag may be ignored if the server already sets X-Frame-Options headers.

Restart your server and reload the slide after making changes.
